jQuery Resume

Projects \ Online Resume

Resume made with JavaScript and jQuery

The objectiveof this project was to familarize students with jQuery and JavaScript by using JSON and jQuery to make elements appear on the screen.

Required Process:

Build four JSON objects, each one representing a different resume section. Make sure SON objects are formatted correctly using JSONlint.com.

1.Iterate through each JSON object and append its information to index.html in the correct section.

2. Use JavaScript method string.replace(old, new) to swap out all the placeholder text (e.g. %data%) for data from your resume JSON objects.

3. Add interactive map

4. All of code for adding elements to the resume should be within functions. And all functions should be encapsulated within the same objects containing resume data. For instance, functions for appending work experience elements to the page should be found within the same object containing data about your work experience.

5. Resume should also console.log() information about click locations.

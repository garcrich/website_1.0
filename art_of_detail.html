ART OF DETAIL

Projects \ Art of detail

Static Website

This project was for an auto detailing company in Hawaii. The company needed a very simple website to display contact information and services. The colors they preferred were white and black.

Goals

My objective was to only used HTML and CSS as to not be distracted by JavaScript.

Reflections

Among the biggest challenges was learning how to take technologies learned in lessons and translating them into an actual project. This meant that a lot of time was spent reading blogs posts, scanning stack overflow, and reviewing course previously taken, to improve my knowledge of HTML and CSS.

The value of planning was made apperent looking back. Skectching wireframes for the entire website for all views may have taken some time, but the time lost because of disorganization was much greater

Making a point to not use JavaScript was a mistake as JavaScript is best suited for interaction. br Eventually I did use jQuery for an accordion like drop down menu on the services tab.

It was made obvious that I needed to organize my CSS file and provide better class names for future projects. Also, the importance of a grid frame work was also made apparent.

Another area for improvment was graphic design. While the web site was functional it seemed to be lacking originality and order.

How to Interact

This is a standard website taht is fully responsive. If on a phone or tablet, switch from portrait view and landscape mode to see changes in layout. If you are using a desktop, shrink and expand your browser to see the different layouts.
